Pokemon Showdown Showcase

Built using HTML 5, CSS, JS, and SQL

This was the second website that I ever worked on and it was a group project with Case Zumbrum and Louis Li. The goal of the site was to create a place for people interested in playing competitive Pokemon on Pokemon Showdown to see what the most popular pokemon by types are based off of the past 20,000 matches played on the site which we access through Pokemon Showdown's API. If you want to take a look then you can by clicking the link above.

Takeaways

This was certainly a learning experience and was great for me to understand the fundamentals of web development, but the end product has not aged well. Still, it was vital in helping me learn from my mistakes with the layout of the site and the user interface to build more appealing sites in the future like Gator Guide or this portfolio website.
